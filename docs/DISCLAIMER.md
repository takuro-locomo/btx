# 免責事項・医療広告ガイドライン対応

## アプリ内に常時表示すべき免責文（ja）

```
本シミュレーターは教育目的のツールであり、医療行為の代替ではありません。
表示される効果・副作用は文献データに基づく確率的予測であり、
実際の治療結果は個別の解剖・既往により異なります。
治療判断は必ず医師の診察を受けてください。
```

## 英語版

```
This simulator is for educational purposes only and does not replace medical care.
Predicted outcomes and side effects are probabilistic estimates based on published evidence;
actual results vary by individual anatomy and history.
Always consult a qualified physician for treatment decisions.
```

## 日本の医療広告ガイドライン上の留意点（厚労省）

このアプリは「医療機関の HP」ではなく「教育シミュレーター」だが、
クリニックがプロモーションに使う場合、以下に注意：

### NG 表現
- 「必ず治る」「100%安全」など効果断定
- 「日本一」「最高」など比較優良
- 「No.1」「業界初」（実証データなしの場合）
- 加工・修正したビフォーアフター
- 治療内容・期間・費用・主なリスクを明記しない症例提示

### OK 表現
- 「〇〇のしわ改善が期待できます」
- 「文献では X% の患者で改善」（出典明記）
- 症例ベースの解説（条件全部明記）

### 必須記載（クリニック導入時）
- 治療内容
- 標準的な治療期間・回数
- 概算費用
- 主なリスク・副作用
- 医師名・経歴

## YMYL（Your Money or Your Life）対応

検索エンジン上で医療系コンテンツとして扱われるため：
- 一次ソース（論文・添付文書）への直接リンク
- 監修医師の明記
- 更新日時の表示
- 著者・運営者情報の明記
- E-E-A-T（経験・専門性・権威性・信頼性）を担保

## アプリ実装上のチェックリスト

- [ ] フッターに常時免責バナー
- [ ] 初回起動時にモーダルで免責同意（任意）
- [ ] すべての副作用説明に出典を併記
- [ ] 「効果が出る」ではなく「効果が期待できる」表現
- [ ] 数値は「文献上 X%」と出典付き
- [ ] 監修医師欄を設置可能（クリニックロゴと並列）
- [ ] 利用規約・プライバシーポリシーへのリンク

## 実装すべきコンポーネント

- `<DisclaimerBanner />` — 常時表示フッター
- `<EvidencePanel />` — 各副作用に出典展開
- `<ConsentModal />` — 初回利用時の同意（オプション）
- `<MedicalSupervisorBadge />` — 監修医師表示
