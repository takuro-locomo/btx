## 1. 基本理論

### 1-1. 表情筋（Mimetic muscles）とは

頭頸部の筋肉のうち、皮膚に直接付着して表情を作る筋群。骨と骨をつなぐ普通の骨格筋と違い、「骨から皮膚」または「皮膚から皮膚」に付くのが特徴。

→ だからこそ、収縮するたびに「皮膚そのものを引っ張る」ことになり、長年繰り返すうちに皮膚に折り目（シワ）が固定する。

### 1-2. シワができる3段階

| 段階 | 状態 | 介入 |
|---|---|---|
| 動的シワ（dynamic） | 表情を作った時だけ出る | ボトックス○、最も適応 |
| 静的シワ（static） | 無表情でも残る | ボトックス△＋ヒアルロン酸など併用 |
| 皮膚老化シワ | コラーゲン減少などによる | レーザー・ピーリング系 |

→ ボトックスは原則「動的シワ」が最良適応。ただし長期治療で静的シワが薄くなる例も多い。

### 1-3. なぜボトックスでシワが消えるのか

ボトックスは神経筋接合部でアセチルコリン放出を阻害する。結果：

```
表情筋が動かない → 皮膚を引っ張らない → 折り目が形成されない → シワが伸びる
```

ポイント：シワそのものを消すのではなく、シワを作る筋肉を一時的に止めるということ。

### 1-4. 「シワに対して垂直な筋走行」が原則

シワはそれを作る筋線維と直交する方向にできる。

| シワの方向 | 原因筋の走行 | 例 |
|---|---|---|
| 横じわ | 縦走筋（垂直線維） | 額の横じわ ← 前頭筋（縦走） |
| 縦じわ | 横走筋（水平線維） | 眉間縦じわ ← 皺眉筋（水平） |
| 斜めじわ | 斜走筋 | バニーライン ← 鼻筋斜部 |
| 放射状じわ | 輪状筋 | 目尻 ← 眼輪筋（リング） |

→ 筋走行を理解すれば、シワの原因筋が逆算できる。

## 2. 筋肉マップ

各筋について、以下の構造で記載：

- **解剖**: 起始・停止・作用
- **作るシワ**: 動的・静的の両方
- **ボトックス治療**: 効果・標的部位
- **拡散リスク（隣接筋）**: 注入時に拡散すると影響する筋

### 2-1. 上部顔面

#### 🔵 前頭筋 (Frontalis) — 額の横じわ

```
解剖:
  起始: 帽状腱膜（頭頂）
  停止: 眉毛の皮膚・皮下組織
  作用: 眉毛を挙上する（"驚き"の表情）
        → 眉毛挙上の唯一の筋
  神経: 顔面神経 側頭枝
```

**作るシワ**: 額の横じわ（horizontal forehead lines）。1〜5本の横線が額に走る。

**ボトックス治療**:
- 標的: 前頭筋全体（4〜8 ポイント、合計 10–20 U）
- 絶対鉄則: 必ず眉間（皺眉筋・鼻根筋）とペアで打つ。前頭筋単独抑制は眉毛下垂を招く（前頭筋は眉挙上の唯一の筋であり、抑制すれば眉が必ず下がる）

**拡散リスク**:
- 下方拡散 → 自分自身の下部抑制 → 眉毛下垂
- 内側拡散 → 眉間複合筋への影響（通常はむしろ望ましい併用）

重要なバランス：
```
前頭筋 (挙上) ⇄ 眉間複合体 (下制)
       ↑              ↑
   この拮抗を崩すと眉毛位置が乱れる
```

---

#### 🔵 皺眉筋 (Corrugator supercilii) — 眉間の縦じわ

```
解剖:
  起始: 前頭骨 眉間部
  停止: 眉毛中央〜外側 1/3 の皮膚
  作用: 眉毛を内下方に引く（"しかめっ面"）
  神経: 顔面神経 側頭枝
```

**作るシワ**: 眉間の縦じわ（"11" サイン、glabellar lines）。皺眉筋は左右に水平方向の線維があり、それが収縮すると皮膚を中央に引き寄せる → 縦じわ。

**ボトックス治療**:
- 標的: 左右各 2 ポイント（内側・外側）、各 4 U
- 内側ポイント: 眉毛内端の上
- 外側ポイント: 眼窩上縁から 1cm 以上上（FDA-PI 警告）

**拡散リスク**:
- 下方／後方拡散 → 上眼瞼挙筋 (LPS) → 眼瞼下垂（最重要合併症）
- 上方拡散 → 前頭筋 → 額の動き低下（軽度）

「外側 corrugator」が拡散の鬼門：眼窩上縁の直上を打つと、上眼瞼挙筋への拡散が起こり眼瞼下垂を招く。FDA-PI で明記されている警告ゾーン。

---

#### 🔵 鼻根筋 (Procerus) — 眉間中央の横じわ

```
解剖:
  起始: 鼻骨・鼻軟骨（下方）
  停止: 眉間中央の皮膚（上方）
  作用: 眉間中央を下に引く（"怒り"の表情）
  神経: 顔面神経 頬骨枝（一部 側頭枝）
```

**作るシワ**: 鼻根の横じわ（鼻根のあたり、眉間中央の水平方向）。鼻根筋は縦走なので、収縮すると横じわを作る（基本則：筋走行と直交）。

**ボトックス治療**:
- 標的: 中央 1 ポイント、4 U
- 眉間 5 点法の中央点に該当（FDA-PI 標準）

**拡散リスク**:
- 通常は安全な部位（重要な隣接構造から離れている）
- 過量で前頭筋下部・皺眉筋への影響

眉間注入のしわパターン分類：
- U型 = procerus 主体 → 中央寄り注入で効果大
- V型 = corrugator 主体 → 内側 corrugator 重視
- オメガ型 = 両者の複雑な組み合わせ

（NEWLUX2024 韓国コンセンサス）

---

#### 🟢 眼輪筋 (Orbicularis oculi) — 目尻のカラスの足跡

```
解剖:
  起始/停止: 眼瞼内側靭帯〜涙嚢〜眼瞼外側靭帯
            （リング状に眼を取り囲む）
  部位:
    - 眼窩部 (orbital part)：外側、目を強く閉じる
    - 眼瞼部 (palpebral part)：上下瞼、まばたき
      - 隔膜前部 (preseptal)
      - 瞼板前部 (pretarsal)
  作用: 閉瞼・涙液ポンプ
  神経: 顔面神経 側頭枝・頬骨枝
```

**作るシワ**:
- 目尻のカラスの足跡（crow's feet、外側放射状）
- 下眼瞼の横じわ（jelly roll）— 笑った時に下瞼の盛り上がり
- 上眼瞼のシワ

→ 眼輪筋はリング状なので、外側ではリングが外向きに開いて引く形になり、放射状のシワが生じる。

**ボトックス治療**:
- カラスの足跡: 外眼角から 1〜1.5cm 外側、上下に C 字状に 2〜3 ポイント、片側 7.5–15 U
- 浅層注入（皮内〜浅筋層 1–2mm）

**拡散リスク**:
- 眼瞼部（palpebral part）への拡散: 下眼瞼下垂・閉瞼力低下・ドライアイ悪化
- 頬骨筋への下方拡散: 大頬骨筋に近すぎる注入（外眼角の下方 1cm 以内）→ 頬部下垂・笑顔の非対称
- 外直筋への深部拡散（極稀）: 複視

重要原則：眼窩外側骨縁の外側にのみ注入。骨縁の内側（眼窩内）は厳禁。

### 2-2. 中部顔面

#### 🟠 鼻筋 (Nasalis) — バニーライン

```
解剖:
  起始: 上顎骨 犬歯窩
  停止: 鼻軟骨〜対側の鼻筋線維と連続
  部位:
    - 横部 (transverse, compressor naris)：鼻孔を狭める
    - 翼部 (alar, dilator naris)：鼻孔を広げる
  作用: 鼻翼の動き、鼻翻し（"嫌悪"の表情）
  神経: 顔面神経 頬骨枝
```

**作るシワ**: バニーライン（鼻根の側方斜めじわ）。鼻を翻して笑った時に、鼻側面に「ウサギのヒゲ」状のシワができる。

**ボトックス治療**:
- 標的: 鼻骨側方斜面、鼻側壁の上 2/3 部分
- 左右各 1 ポイント、2–4 U
- 浅く注入

**拡散リスク**:
- 下方拡散 → 上唇鼻翼挙筋 (LLSAN) → Joker smile（上唇片側挙上）
- 鼻翼基部より下に打たないこと

併用ポイント：眉間ボトックス後にバニーラインが目立つようになるケースが多い（眉間ヘの代償）。

---

#### 🔴 上唇鼻翼挙筋 (LLSAN: Levator labii superioris alaeque nasi) — ガミースマイル

```
解剖:
  起始: 上顎骨前頭突起（眼窩内側縁付近）
  停止: 上唇および鼻翼の皮膚
  作用: 上唇挙上 + 鼻翼挙上（"歯を見せる笑顔"）
  神経: 顔面神経 頬骨枝
```

**作るシワ・問題**:
- ガミースマイル: 笑った時に歯茎が見えすぎる（LLSAN の過収縮）
- 鼻唇溝（laughter lines）の深化に寄与

**ボトックス治療**:
- 標的: Yonsei point（鼻翼外側基部・鼻唇溝・人中側方を結ぶ三角の中心）
- 左右各 1 ポイント、1–2.5 U
- 浅く（1–2mm）

**拡散リスク**:
- 過量 → 上唇下垂、笑顔がぎこちなく
- 外側拡散 → 上唇挙筋・小頬骨筋 → 笑顔の非対称
- 「長い上唇」に見える美的副作用

慎重要件：左右対称性が極めて重要。1U の差でも非対称が出る。

---

#### 🔴 上唇挙筋 (Levator labii superioris) — 鼻唇溝

```
解剖:
  起始: 眼窩下縁
  停止: 上唇の皮膚
  作用: 上唇を挙上
  神経: 顔面神経 頬骨枝
```

**作るシワ**: 鼻唇溝（nasolabial fold）の形成への寄与。上唇を挙げる動作で頬から口角に向かう線が深くなる。

**ボトックス治療**:
- 鼻唇溝はボトックス単独では適応外（皮下脂肪・骨吸収などが主因）
- 通常はヒアルロン酸併用が標準
- ボトックスは LLSAN や大頬骨筋とのバランス調整目的で限定的に

---

#### 🔴 大頬骨筋 (Zygomaticus major) — 笑顔の主役

```
解剖:
  起始: 頬骨弓 外側部
  停止: 口角（modiolus）
  作用: 口角を上外側に引く（"笑顔"の中心）
  神経: 顔面神経 頬骨枝
```

**作るシワ**:
- 笑った時の頬上の盛り上がり
- 頬骨皮膚しわ（laugh lines、頬の上）
- 鼻唇溝の上部

**ボトックス治療**:
- 基本的にボトックス標的にしない（笑顔そのものを減らしてしまう）
- ただし、エラ（咬筋）注入時の意図せぬ拡散先として最重要警戒筋
- 表在注入で大頬骨筋が抑制されると、笑顔の左右非対称が起こる

---

#### 🔴 小頬骨筋 (Zygomaticus minor) — 補助挙上

大頬骨筋の内側にある細い筋。上唇外側を挙上する補助筋。

ボトックス標的としては通常扱わないが、LLSAN 注入時の隣接筋として認識する必要がある。

---

#### 🟠 笑筋 (Risorius) — 横方向の口角引き

```
解剖:
  起始: 咬筋筋膜
  停止: 口角の皮膚
  作用: 口角を**水平外側**に引く（"嘲笑"の表情）
  神経: 顔面神経 下顎縁枝
```

**作るシワ**: 口角横の細かいしわ。

**ボトックス治療**:
- 通常標的にしない
- 咬筋注入時の拡散先として警戒すべき筋（咬筋前縁の前方に打つと笑筋に拡散 → 笑顔非対称）

---

### 2-3. 下部顔面

#### 🔴 口輪筋 (Orbicularis oris) — 口周りの縦じわ

```
解剖:
  起始/停止: 口の周囲をリング状に取り囲む
            （頬筋、その他の表情筋線維と連続）
  作用: 口を閉じる、唇をすぼめる、発音
  神経: 顔面神経 頬骨枝・下顎縁枝
```

**作るシワ**:
- 口周りの縦じわ（perioral lines、バーティカルリップライン）
- スモーカーズライン（喫煙習慣で深化）
- 上唇の縦じわ（特に上唇）

→ 口輪筋は輪状なので、収縮すると唇の周りに放射状の縦じわができる。

**ボトックス治療**:
- 標的: 唇の赤縁直上（vermilion border）に皮内注入
- 4 ポイント（上下唇に各 2）、各 1–2 U、合計 4–8 U
- リップフリップ目的: 上唇のみに 2–4 U

**拡散リスク（深刻）**:
- 発音障害（"p", "b", "m"）— 用量依存、過量で確実
- ストロー使用困難・飲み物こぼす
- 口角下制筋への拡散 → 笑顔非対称
- 口唇閉鎖不全

絶対禁忌: 口角から 5mm 以内には絶対に打たない。1ポイント 2U を超えない。

---

#### 🟣 オトガイ筋 (Mentalis) — 顎の梅干しじわ

```
解剖:
  起始: 下顎骨 切歯窩
  停止: 顎の皮膚
  作用: 下唇を前突・挙上（"考え込む"表情）
        → 過収縮で顎の皮膚に「梅干し」状の凹凸
  神経: 顔面神経 下顎縁枝
```

**作るシワ**:
- 顎の梅干しじわ（chin dimpling、cobblestone chin）
- 顎の横じわ

**ボトックス治療**:
- 標的: 正中 1 ポイント（下顎骨前面、最も突出した点）
- 4–10 U（標準 5 U）
- 必ず正中に留める

**拡散リスク**:
- 外側拡散 → 下唇下制筋 (DLI) → 下唇のコントロール障害、非対称な唇の動き
- 下唇前突感の喪失

絶対禁忌: 正中から 1cm 以上外側には打たない。

---

#### 🟣 口角下制筋 (DAO: Depressor anguli oris) — マリオネットライン

```
解剖:
  起始: 下顎骨 斜線
  停止: 口角（modiolus）
  作用: 口角を下方に引く（"悲しみ"の表情）
  神経: 顔面神経 下顎縁枝
```

**作るシワ**:
- マリオネットライン（口角から顎に向かう縦線）
- "への字口"の原因筋

**ボトックス治療**:
- 標的: 下顎縁、口角から下 1cm × 外側 1cm の点（"jowl line"上）
- 左右各 2–5 U
- 十分に外側に打つ

**拡散リスク**:
- 下唇下制筋 (DLI) への拡散 → 下唇の非対称、笑顔の歪み（最頻合併症）
- 口角の左右差

絶対禁忌: 口角直下、内側寄り（DLI 領域）に打たない。

---

#### 🟣 下唇下制筋 (DLI: Depressor labii inferioris) — 下唇の動き

```
解剖:
  起始: 下顎骨 斜線（DAO のすぐ内側）
  停止: 下唇の皮膚と粘膜
  作用: 下唇を下外方に引く（"不機嫌"・笑顔の下唇展開）
  神経: 顔面神経 下顎縁枝
```

**ボトックス治療**:
- DLI 自体を標的にすることは少ない
- DAO 注入・mentalis 注入時の拡散先として最重要警戒筋
- DLI が抑制されると下唇の動きに左右差が出て、笑顔が歪む

### 2-4. 咀嚼・首

#### 🔵 咬筋 (Masseter) — エラ・小顔

```
解剖:
  起始: 頬骨弓
  停止: 下顎角・下顎枝
  作用: 下顎挙上（噛む）
  神経: 三叉神経 下顎神経
  ※ 唯一の咀嚼筋（表情筋ではない）
```

**問題**: 咬筋肥大によるエラの張り（masseter hypertrophy）。表情筋でなくシワを作るわけではないが、輪郭リシェイピング目的でボトックスが使われる。

**ボトックス治療**:
- 標的: 下顎角を起点とした安全三角形内
- 片側 25–50 U、3 ポイント、深部筋層（5–10mm 深）

**拡散リスク**:
- 咬筋前縁の前方 → 笑筋・大頬骨筋・頬筋 → 笑顔非対称
- 表在注入 → 浅層線維のみ抑制 → 逆説的膨隆（深層代償肥大）または頬部陥没（"hourglass deformity"）
- 咀嚼力低下（用量依存、4–6 週で回復）

---

#### ⚫ 広頸筋 (Platysma) — 首のたるみ・ネックバンド

```
解剖:
  起始: 鎖骨下・大胸筋上部の皮下組織
  停止: 下顎縁・口角・頬下部の皮膚
  作用: 下顎・口角を下方に引く、首の皮膚を引き締める
  神経: 顔面神経 頸枝
  ※ 表情筋でありながら首にかかる広い薄い筋
```

**作るシワ・問題**:
- ネックバンド（platysmal bands、首の縦すじ）
- 下顎輪郭の不明瞭化（jowl の悪化）
- マリオネットラインへの寄与

**ボトックス治療（"Nefertiti lift"）**:
- 標的: バンド本体（縦走線維）+ 下顎縁付近
- 総量 25–60 U、バンドあたり 10–20 U（3–6 ポイント）

**拡散リスク（重大）**:
- 嚥下困難 (dysphagia) — 深部注入で発生、重大合併症
- 頸部筋力低下
- 発音困難

上級者向け部位：アプリでは医師専用モードで扱う。

## 3. 逆引き表

「このシワを治療したい」→「どの筋を狙うか」逆引きできる。

| シワ | 主犯筋 | 共犯筋 | 標的部位 (zone) |
|---|---|---|---|
| 額の横じわ | 前頭筋 | — | `forehead` |
| 眉間の縦じわ "11" | 皺眉筋 | 鼻根筋、眉毛下制筋 | `glabella` |
| 鼻根の横じわ | 鼻根筋 | — | `glabella`（中央点） |
| 目尻のカラスの足跡 | 眼輪筋（外側） | — | `crows_feet_left/right` |
| 下眼瞼じわ (jelly roll) | 眼輪筋（眼瞼部） | — | （上級、低用量） |
| バニーライン（鼻側方斜め） | 鼻筋 | — | `bunny_lines` |
| ガミースマイル | LLSAN | 上唇挙筋・小頬骨筋 | `gummy_smile` |
| 口周り縦じわ | 口輪筋 | — | `lip_flip` / perioral |

| 顎の梅干しじわ | オトガイ筋 | — | `mentalis` |

## 4. 治療マッピング

「ここに打つと、ここの筋が緩み、このシワが消える」の流れ：

```
注入点 (zone)              標的筋               緩む動き                   消えるシワ
─────────────────         ─────────         ──────────────────────       ───────────────
forehead (額)        →    前頭筋        →    眉毛挙上抑制         →    額の横じわ
glabella (眉間)      →    皺眉筋        →    眉内下方引き抑制     →    眉間縦じわ
                          鼻根筋        →    眉間中央下方引き抑制 →    鼻根横じわ
crows_feet (目尻)    →    眼輪筋外側部  →    閉瞼力低下           →    カラスの足跡
bunny_lines          →    鼻筋          →    鼻翻し抑制           →    バニーライン
gummy_smile          →    LLSAN         →    上唇過挙上抑制       →    ガミースマイル
lip_flip             →    口輪筋        →    唇のすぼめ抑制       →    口周り縦じわ
mentalis (顎)        →    オトガイ筋    →    顎の収縮抑制         →    梅干しじわ
dao (口角)           →    DAO           →    口角下方引き抑制     →    マリオネットライン
masseter (エラ)      →    咬筋          →    咀嚼力減少 → 萎縮    →    エラの輪郭縮小
platysma (首)        →    広頸筋        →    首皮膚下方引き抑制   →    ネックバンド
```

## 5. シワを作る筋肉の組み合わせ

シワは1筋単独でなく複数筋の協働で作られることが多い。これがアプリの「複合的しわ表示」の根拠：

### 5-1. 眉間の "11" サイン

```
皺眉筋（左右、水平線維）   →    皮膚を中央に寄せる
        +
鼻根筋（中央、垂直線維）    →    眉間中央を下に引く
        +
眉毛下制筋・眼輪筋上内側    →    眉毛を下げる
        =
深い縦じわ "11" + 鼻根の横じわ
```

→ だから FDA-PI 標準は「皺眉筋 4点 + 鼻根筋 1点 = 5点法」。

### 5-2. 笑顔のシワ群

```
眼輪筋外側 → カラスの足跡
大頬骨筋   → 頬上の盛り上がり線
LLSAN     → 鼻唇溝の上部
口輪筋    → 唇の動き
       =
笑顔の総合的なシワパターン
```

→ 「笑顔のシワを全部消す」は美的に NG。眼輪筋（カラスの足跡）だけを慎重に標的に。

### 5-3. 老け顔の下半分

```
DAO         → 口角下垂（"への字"）
広頸筋      → 下顎縁の不明瞭化
咬筋肥大    → エラの張り
       =
「不機嫌そう」「老けて見える」下顔面
```

→ "Nefertiti lift" は DAO + 広頸筋下顎縁を組み合わせた治療コンセプト。

## 6. アプリ実装上の応用

### 6-1. しわパス (Wrinkle Path) と zone の対応

`src/types/botox.ts` の `Wrinkle.treatedByZones` は、このドキュメントの「逆引き表」と一致させる：

```typescript
{
  id: "glabellar_lines",
  paths: ["M ... 縦じわ左", "M ... 縦じわ右"],
  treatedByZones: ["glabella"]  // glabella zone を治療すると消える
}
```

### 6-2. 副作用判定ロジックの根拠

各 `dangerZone.effect` は「この筋への意図せぬ拡散」を意味する：

```typescript
// glabella zone の例
dangerZones: [
  {
    polygon: <眉毛中央上1cm内>,
    effect: "eyelid_ptosis",       // ← LPS への拡散を表現
    description: "皺眉筋外側からの上眼瞼挙筋への拡散リスク"
  }
]
```

→ 副作用 ID は「拡散先の筋が抑制された結果起きる現象」と対応する。

### 6-3. 「適切な配置」の検証

`zones.ts` の `idealPoints` は本書の解剖学的記述から引き出された具体座標：

```
皺眉筋（水平線維）→ 内側点 + 外側点
       ↓
       「左 corrugator 内側」「左 corrugator 外側」の2点
       「右 corrugator 内側」「右 corrugator 外側」の2点
       「procerus 中央」の1点
       = FDA-PI 5点法
```

### 6-4. 教育用説明テキストの素材

各 zone の `description` フィールドに、本書の「作るシワ」「ボトックス治療」セクションの内容を要約して入れる。
