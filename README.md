# ボトックスシミュレーター

エビデンスベースの A 型ボツリヌス毒素（ボトックス）注入シミュレーター。
投与量・部位・しわのタイプを入力すると、改善・副作用がイラストの変化で可視化される教育ツール。

## 目的

- 美容皮膚科スタッフ研修
- 医師トレーニング・症例検討
- 患者カウンセリング補助
- 一般教育

## 特徴

- **エビデンスベース**: FDA 添付文書、Carruthers コンセンサス、2024 国際/韓国コンセンサス論文を直接引用
- **失敗事例も学べる**: 過量・誤位置で「変な顔になる」を視覚化
- **副作用予測**: 眼瞼下垂・眉毛下垂・Spock brow など13種の副作用を確率表示

## 開発

```bash
npm install
npm run dev
```
